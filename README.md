# Skills

- ilspy-decompile
  Decompiles .NET assemblies (`.dll`/`.exe`/`.netmodule`/`.winmd`/`.nupkg`) to readable C# via `ilspycmd` before analysis.
  Requires the .NET SDK (9.0+ recommended). The skill will install `ilspycmd` as a global dotnet tool on first run (`dotnet tool install -g ilspycmd`).
  Claude Code: install by downloading `zips/ilspy-decompile.zip` and extracting it to `~/.claude/skills/ilspy-decompile/`.
